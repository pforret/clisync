# clisync

Backup to local disks/S3 cloud storage with daily/weekly/monthly (CLI bash)

## Installation

with [basher](https://github.com/basherpm/basher)

	$ basher install pforret/clisync

or with `git`

	$ git clone https://github.com/pforret/clisync.git
	$ cd clisync

## Usage

	> clisync.sh [action]
